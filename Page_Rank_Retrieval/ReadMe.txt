                                              How to run
===========================================================================================================
1) Unzip given file
2) Make sure you have python installed on your pc. I have used 3.6.0 version
3) Open terminal
4) Excute below commands
    a) python pagerank.py
    b) Enter the input file path. Example: "/Users/HarshPatil/CS429/Assignment_4_Page_Rank/test3.txt"
    c) Enter the output file path. Example: "/Users/HarshPatil/CS429/Assignment_4_Page_Rank/output.txt"
    d) Top 10 page ids along with it's rank will be displayed and rest will be written in output.txt
